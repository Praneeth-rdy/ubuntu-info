******Installing Driver******
(Better to use venv while installing)

1) Python needs a MySQL driver to access the MySQL database.

2) Recommended to use PIP to install "MySQL Connector".

3) Type this on command line - ' python -m pip install mysql-connector-python '

4) Test MySQL Connector by importing it.

5) Use the environment in which you have installed it else it won't work.
